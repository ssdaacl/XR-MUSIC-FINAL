An minimalist, interior-magazine inspired web application for curating and playing local high-fidelity audio (WAV). 

## Features
- **Minimalist Aesthetic**: High-end neutral palette with a magazine-style intro.
- **Dynamic Categorization**: Automatically groups tracks based on filenames (`Title_Feature1_Feature2.wav`).
- **Local Curation**: Browse entire local folders with privacy and speed.
- **Custom Visuals**: Procedurally generated gradients for every track.
- **Master Download**: Direct access to source files.

## Technical Stack
- **Framework**: React 19
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Typography**: Cormorant Garamond (Serif), Inter (Sans)
- **Deployment**: Vercel

## How to use locally
1. Clone the repository.
2. Run `npm install`.
3. Start development server with `npm run dev`.
